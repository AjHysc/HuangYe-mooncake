# 荒野月饼
增加了五种月饼

##制作
小麦种子=油

小麦=面粉

面粉+油+糖=面皮

面皮×9=月饼团

面皮×6+苹果×3=苹果月饼团

面皮×6+土豆×3=莲蓉月饼团

面皮×6+土豆×2+鸡蛋=蛋黄月饼团

面皮×6+下界合金碎片+钻石+绿宝石=五金月饼团

任意月饼团=生月饼

任意生月饼=熟月饼(熔炉、烟熏炉、篝火)

## 效果
五金月饼:食用时长60s，获得伤害+抗性4*20分钟

苹果月饼：生命提升30s

蛋黄月饼:生命恢复30s

莲蓉月饼：急迫255*5s

月饼：反胃60s

所有生月饼：饥饿255*60s

(熟月饼均回复20点饱食度)
